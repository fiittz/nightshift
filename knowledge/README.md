# Knowledge Base

Drop files here that agents should reference. Supports:
- `.md` — Markdown documents
- `.txt` — Plain text
- `.json` — Structured data
- `.pdf` — PDF documents (text extracted)

Files in this directory are automatically included in agent context when relevant.

## Organization
- `company/` — Company info, brand guidelines, tone of voice
- `product/` — Product docs, specs, architecture decisions
- `domain/` — Domain-specific knowledge (tax law, regulations, etc.)
- `customers/` — Customer profiles, case studies, FAQs
- `competitors/` — Competitive analysis, market research
