# Budget Tracker

A simple Python application for tracking expenses and generating budget reports.

## Features

- Set monthly budget
- Add expenses with categories
- View all expenses
- Generate reports in multiple formats (text, CSV, JSON)
- Export expenses to CSV
- Calculate remaining budget
- Track expenses by category

## Files

- `budget_tracker.py` - Main application with BudgetTracker class
- `example_usage.py` - Example demonstrating how to use the tracker
- `requirements.txt` - Python requirements (none required - uses stdlib)
- `budget_data.json` - Data file created when using the tracker
- `budget_report.*` - Report files generated by the system
- `expenses_export.csv` - Exported expenses file
